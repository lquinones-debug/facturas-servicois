App "Control de Facturas de Servicios" — archivos para subir a GitHub
=====================================================================

QUE ES ESTA CARPETA
-------------------
Son los archivos de la aplicacion web de Control de Facturas de Servicios
(la de facturas-indian.streamlit.app). Es la COPIA LIMPIA (sin claves ni
datos privados) que se sube al repositorio de GitHub para actualizar la app.

COMO SE USA (cuando Claude te avise que actualizo algo)
-------------------------------------------------------
1. Entra a GitHub -> repositorio "facturas-servicois".
2. Add file -> Upload files -> arrastra los archivos que Claude te indique
   (normalmente app.py y sheet_db.py) -> Commit changes.
3. En Streamlit Cloud: Manage app -> boton de tres puntitos -> Reboot app,
   y despues Ctrl+F5 en el navegador.
   (El Reboot es IMPORTANTE: si no, la app sigue usando la version vieja.)

QUE NO HACE FALTA
-----------------
- No tenes que abrir ni editar estos archivos a mano (de eso se encarga Claude).
- Este LEEME no se sube a GitHub.
